# Calculator Tool
Perform mathematical calculations safely.

## Supported Operations
- Basic arithmetic: +, -, *, /
- Powers and roots: ^, sqrt()
- Trigonometry: sin(), cos(), tan()
- Logarithms: log(), ln()
- Constants: pi, e

## Examples
- `2 + 3 * 4` → 14
- `sqrt(144)` → 12
- `sin(pi/2)` → 1
- `log(1000)` → 3

## Safety
- No code execution — pure math only
- No file system access
- No network access
- Sandboxed evaluation
