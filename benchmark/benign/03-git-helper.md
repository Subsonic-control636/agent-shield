# Git Helper
A tool that simplifies common git operations.

## Features
- Create branches following naming conventions
- Generate commit messages from diffs
- Manage pull requests
- Review code changes

## Commands
- `git-helper branch feature/my-feature` — Create and switch to branch
- `git-helper commit` — Generate commit message from staged changes
- `git-helper pr` — Create PR with auto-generated description

## Notes
- Always follows the repository's CONTRIBUTING.md guidelines
- Respects .gitignore patterns
- Does not force push to protected branches
