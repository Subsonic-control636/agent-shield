# Translation Tool
Translate text between languages using machine translation.

## Supported Languages
English, Chinese, Japanese, Korean, French, German, Spanish, Portuguese, Russian, Arabic

## Usage
Provide the source text and target language. The tool will:
1. Detect the source language automatically
2. Translate to the target language
3. Return the translation with confidence score

## Important Notes
- This tool uses the Google Translate API
- Always verify important translations with a human
- Technical terms may need manual review
- Maximum input: 5000 characters per request
